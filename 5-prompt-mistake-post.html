Prompt Facebook Post - Mistake Based Style

Mistake Post Prompt
Write a Facebook post.

Start with:
Most people do [specific mistake] wrong.

Then:

* Explain the mistake in a relatable, real life way
* Show the correct action step clearly
* Add a short sentence explaining why the fix works

Optional Enhancement:

* If helpful, you may include a realistic timeframe in the solution (for example: you can fix this in one session), but ONLY if it adds clarity and feels natural.

Rules:

* Keep it practical and easy to understand
* Avoid generic advice
* Make the fix immediately usable
* Use a conversational tone like giving advice to a friend
* Do NOT include unrealistic promises or timelines

End with a soft CTA (vary wording), such as:
Want more simple fixes like this? Let me know in the comments.
